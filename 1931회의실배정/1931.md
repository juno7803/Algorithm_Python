## 회의실 배정
> 분류 : Greedy

lambda의 사용
```python
    ele = sorted(ele, key=lambda x: x[0]) # a(첫 번째 원소로 정렬)
    ele = sorted(ele, key=lambda x: x[1]) # b(두 번째 원소로 정렬)
    # ele.sort(key=lambda x: (x[1],x[0])) # 와 같이 한번에 정렬 가능
```

이차원 배열에 대해 다음과 같이 순서쌍으로 for문 사용 가능
```python
    for i,j in ele:
        if i >= end:
        cnt += 1
        end = j
```
